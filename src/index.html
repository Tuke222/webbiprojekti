Tervetuloa verkkosivuprojektiin

Tämä on yksinkertainen staattinen verkkosivu, joka on tehty osana Linux-, Git- ja HTML/CSS-harjoitusta. Sivusto on rakennettu komentoriviltä Ubuntu MATE -ympäristössä.

Projektin tarkoitus

Projektin tavoitteena on harjoitella perusasioita, kuten tiedostojen ja kansioiden käsittelyä Linuxissa, versionhallintaa Gitillä sekä verkkosivujen julkaisua GitHub Pages -palvelussa.

Käytetyt teknologiat

  • Linux ja Bash
  • Git ja GitHub
  • HTML ja CSS

Alla on esimerkkikuva, joka on ladattu internetistä ja upotettu sivulle img-elementillä.
